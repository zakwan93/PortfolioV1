Resume Genius!

BACKGROUND

So, I get this idea after using tons of different SO CALLED FREE RESUME BUILDER which I end up paying money after adding information and design a template.

ABOUT PROJECT

  • Design and developed Ruby on rails application to generate and download a free resume
  • Each user must sign-up to use this app
  • Once user sign-up they can come back anytime and continue working from last stage
  • User must provide the basic infromation needeed for resume to download it
  • User can add extra section in their resume according to their experiance

TECHNOLOGIES

  • Ruby on Rails
  • PostgreSQL database
  • Uikit Framework

EXTRA GEMS

  • devise
  • wicked_pdf
  • wkhtmltopdf-binary
